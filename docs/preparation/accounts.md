# 👤 アカウントの作成

今回の講座で使用するwebサービスのアカウント登録を行います。
登録するアカウントは以下の3種類です。
- **Googleアカウント**(GitHubの登録に使用)
- **GitHub**(作ったホームページを公開するサービス)
- **CodePen**(作ったホームページを保存するサービス)

作品を保存したり、みんなに見せたりするためのアカウントを作りましょう！


# 🙎 アカウントを作る際の注意事項

## 🔐 パスワード管理

### 安全なパスワードの作り方（大文字・小文字・数字等を組み合わせる）
- **良い例**: `MyFav0riteC0lor!sPurple`
- **悪い例**: `password123`

### パスワードの保管
1. **紙に書く**
   - 家の安全な場所に保管

2. **保護者と共有**
   - 忘れたときのために

3. **絶対にしないこと**
   - 友達に教えない
   - ネットに書かない

## 📧 メールアドレスについて

### なぜ必要？
- パスワードを忘れたとき
- アカウントの確認
- 重要なお知らせ

### 注意点
- 迷惑メールフォルダも確認
- 知らないメールは開かない

## 🚨 困ったときは

### ユーザー名が使えない
- 数字を追加してみる
- 別の名前を考える
- ハイフンを使う

### メールが届かない
1. メールアドレスを確認
2. 迷惑メールフォルダを見る
3. もう一度送信してみる

### パスワードを忘れた
1. 「Forgot password?」をクリック
2. メールアドレスを入力
3. メールの指示に従う

## 👨‍👩‍👧‍👦 保護者の方へ

### アカウント作成時のお願い
- お子様と一緒に作成してください
- パスワードは保護者様も把握してください
- 定期的にアカウントの使用状況を確認してください

### プライバシー設定
- 本名や住所は入力しない
- プロフィール写真は実写を避ける
- 公開範囲を適切に設定する

:::tip 💡 ヒント
アカウント名は後から変更できないので、よく考えて決めよう！
:::

## 📧 Googleアカウント

### なぜGoogleアカウントが必要？
- GitHubに簡単にログインできる
- パスワードを覚える必要が少ない
- 安全にアカウント管理ができる

:::warning ⚠️ 重要
Googleアカウント作成は13歳未満は保護者の承諾が必要となります。必ずご一緒に作成して下さい。
:::

### 作り方
1. **Googleアカウント作成ページにアクセス**
   - https://accounts.google.com/signup を開く

2. **必要な情報を入力**
   ```
   姓・名: ニックネームでOK（本名は使わない）
   ```

3. **生年月日と性別**

4. 【Google アカウントへのログインに使用する Gmail アドレスを作成します】の画面で好きなユーザー名を入力


5. **安全なパスワードの作成**
   - 半角アルファベット、数字、記号を組み合わせてパスワードを作成します

4. **電話番号**
   - 13歳未満は保護者の電話番号を入力して下さい。

5. **利用規約に同意**
   - 内容を確認してから同意

## 🐙 GitHubアカウント

### GitHubって何？
GitHubは、作ったウェブサイトを世界中に公開できる場所です！

### 作り方
1. **GitHubにアクセス**
   - https://github.com/login を開く

2. **[Continue with Google] をクリック**
   - [Continue with Google] をクリックし、ソーシャル ログインを使用してサインアップします。

3. **ステップごとに入力**
   
   **Step 1: ユーザー名**
   - Usernameに好きな名前をつける
   ```
   例: sozo-it-lab
   （英数字とハイフンのみ）
   ```

   **Step 2: Githubに関連する情報を受け取る**
   - Email preferences
      - Receive occasional product updates and announcements にチェックをいれるとアップデートのメールが届きます。今回は特に必要ないのでチェックはいりません。

   **Step 3: ユーザー名**
   - Create accountボタンをクリック

4. **アカウントの確認**
   - もしGithubで認証画面が表示された場合、アカウント認証が必要なため、認証コードがメールで届きます。記載されている確認コードをコピーし、Githubの認証画面に貼り付けて認証して下さい。

## 🎨 CodePenアカウント

### なぜアカウントが必要？
- 作った作品を保存できる
- 友達に作品を見せられる
- 他の人の作品も見られる

### 作り方（GitHubアカウントを使う方法）
1. **CodePenにアクセス**
   - https://codepen.io を開く

2. **Sign Upをクリック**
   - 右上の「Sign Up」ボタンを押す

3. **GitHubでサインアップ**
   - 「Sign Up with GitHub」ボタンをクリック
   - 作成したGitHubアカウントでログイン

4. **権限を許可**
   - CodePenがGitHubアカウントにアクセスすることを許可

5. **ユーザー名を設定**
   ```
   ユーザー名: あなたのニックネーム
   （英数字とハイフンのみ）
   ```

:::warning ⚠️ 注意
- 本名は使わない
- GitHubアカウントと連携するので、GitHubを先に作成する
- 保護者と一緒に作成しよう
:::

:::tip 💡 おすすめ
GoogleアカウントやGitHubアカウントでログインすると、パスワードを別に覚える必要がないので便利です！
:::
