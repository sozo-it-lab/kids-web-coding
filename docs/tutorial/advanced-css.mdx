# 🎨 CSSで楽しく装飾しよう

CSSの魔法を使って、ウェブサイトをもっとカラフルで楽しくしましょう！

## 🌈 色をもっと楽しく使おう

### グラデーション（色の変化）

<CodeExample
  title="虹色のボックス"
  html={`<div class="rainbow-box">
  虹色のボックス！
</div>`}
  css={`.rainbow-box {
  background: linear-gradient(to right, red, orange, yellow, green, blue, purple);
  padding: 20px;
  color: white;
  text-align: center;
  border-radius: 10px;
  font-weight: bold;
}`}
/>

## 🎬 アニメーション効果

### 基本のアニメーション

<CodeExample
  title="回転するボックス"
  html={`<div class="spinning-box"></div>`}
  css={`.spinning-box {
  width: 100px;
  height: 100px;
  background-color: #FF6B6B;
  animation: spin 2s linear infinite;
  margin: 20px auto;
}

@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}`}
/>

<CodeExample
  title="脈動するボックス"
  html={`<div class="pulse-box"></div>`}
  css={`.pulse-box {
  width: 100px;
  height: 100px;
  background-color: #4ECDC4;
  border-radius: 50%;
  animation: pulse 1.5s ease-in-out infinite;
  margin: 20px auto;
}

@keyframes pulse {
  0% { transform: scale(1); }
  50% { transform: scale(1.2); }
  100% { transform: scale(1); }
}`}
/>

<CodeExample
  title="跳ねるボール"
  html={`<div class="bouncing-ball"></div>`}
  css={`.bouncing-ball {
  width: 50px;
  height: 50px;
  background-color: #FFE66D;
  border-radius: 50%;
  position: relative;
  animation: bounce 1s ease-in-out infinite;
  margin: 50px auto 20px;
}

@keyframes bounce {
  0%, 100% { 
    transform: translateY(0); 
  }
  50% { 
    transform: translateY(-50px); 
  }
}`}
/>

## 💡 デザインのコツ

### 色の組み合わせ
- 🔵 青 + 🟢 緑 = さわやか
- 🔴 赤 + 🟡 黄 = 元気
- 🟣 紫 + 🔵 青 = 神秘的
- 🟠 オレンジ + 🟡 黄 = 明るい

### 余白を大切に
```css
.good-spacing {
  padding: 20px;  /* 内側の余白 */
  margin: 15px;   /* 外側の余白 */
}
```

### 統一感を出す
- 同じ角の丸み（border-radius）
- 同じ影の強さ
- 同じアニメーション時間
