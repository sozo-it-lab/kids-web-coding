## 📖 ウェブサイトって何？

### 身近な例
みんなが普段見ているもの：
- 🔍 Google（検索サイト）
- 📺 YouTube（動画サイト）
- 🎮 ゲームの公式サイト
- 🏫 学校のホームページ

これらは全部「ウェブサイト」です！

### ウェブサイトの仕組み
```
あなたのパソコン → インターネット → サーバー
     ↑                                    ↓
     ←────────── ウェブサイトのデータ ──────┘
```

## 🎨 どんなサイトが作れる？

### 作品例
- **自己紹介ページ** - 好きなものを紹介
- **趣味のギャラリー** - 写真や絵を展示
- **日記ブログ** - 毎日の出来事を記録
- **ゲーム攻略サイト** - 好きなゲームの情報
- **ペットの紹介** - かわいい写真がいっぱい

## 📝 覚えておきたい用語

| 用語 | 意味 | 例 |
|------|------|-----|
| タグ | HTMLの命令 | `<h1>`, `<p>` |
| 属性 | タグの詳細設定 | `src`, `href` |
| セレクタ | CSSで指定する部分 | `h1`, `.class` |
| プロパティ | CSSの設定項目 | `color`, `font-size` |
