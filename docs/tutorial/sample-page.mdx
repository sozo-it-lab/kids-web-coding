# 🎯 実践サンプル - 自己紹介ページ

チュートリアルで学んだHTML・CSSの技術を使って、実際に完全な自己紹介ページを作ってみましょう！

<SamplePage />

## 🎓 学習のポイント

このサンプルページでは、チュートリアルで学んだ以下の技術をすべて活用しています：

### HTML基礎編で学んだこと
- **セマンティックHTML**: `<header>`, `<section>`, `<footer>`などの適切なタグ使用
- **見出しタグ**: `<h1>`, `<h2>`, `<h3>`の階層構造
- **リスト**: `<ul>`, `<li>`を使ったスキル一覧
- **画像**: `alt`属性付きの適切な画像表示

### CSS基礎編で学んだこと
- **ボックスモデル**: `padding`, `margin`, `border`の使い分け
- **フレックスボックス**: ナビゲーションとカード配置
- **グリッドレイアウト**: レスポンシブなカード配置
- **色とタイポグラフィ**: 読みやすい色選択とフォント設定

### CSS装飾編で学んだこと
- **グラデーション**: 美しい背景とボタン効果
- **シャドウ**: テキストとボックスの立体効果
- **ボーダー**: 様々なスタイルの枠線
- **アニメーション**: ホバー効果とキーフレーム
- **レスポンシブ**: モバイルフレンドリーなデザイン

## 🛠️ 制作プロセス

### 1. 構造設計
```html
<!-- 基本構造 -->
<body>
  <div class="container">
    <header>...</header>
    <section>...</section>
    <footer>...</footer>
  </div>
</body>
```

### 2. コンテンツ作成
- 自己紹介文の作成
- 好きなもの・スキルの整理
- 作品・連絡先情報の準備

### 3. スタイリング
- 色選択（メインカラー + アクセントカラー）
- レイアウト設計（グリッド + フレックス）
- アニメーション追加

### 4. レスポンシブ対応
- モバイル表示の確認
- タブレット表示の調整
- ユーザビリティの向上

## 📝 次のステップ

1. **このコードをダウンロード**して、自分の情報に書き換えてみよう
2. **色やデザインをカスタマイズ**して、オリジナリティを出そう
3. **新しいセクションを追加**して、もっと詳しい自己紹介にしよう
4. **GitHub Pagesで公開**して、みんなに見せよう
